HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag contains information for things like the characters shown on the screen, scale, page dimensions, etc.
  • The section element allows us to divide our page into sections in order to find topics quickly and easily.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS rules follow this format: property: value (for example, color: blue).
  • Colors can be applied using both hexadecimal codes and RGB values.

Git

  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.
  • git add -A: Commits all files to the working branch.
  • git commit -m "insert message here": commits branch to the repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if-else statement allows us to interrupt the control flow based on conditions being met.
  • An array allows us to repeatedly execute a block of code.
